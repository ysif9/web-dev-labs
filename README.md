# Web Development Labs

A collection of lab assignments for the course CSE343 - Web Development, demonstrating progressive web development
concepts.

## Getting Started

Each lab is contained in its own directory. Navigate to the desired lab and follow its specific setup instructions (
typically `npm install` and `npm run dev` for labs with package managers).

## Technologies Used

- HTML5 & CSS3
- JavaScript (ES6+)
- React
- Webpack & Vite (build tools)
